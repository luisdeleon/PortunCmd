# PortunCmd

A Vue 3 admin template built with Vite, Vuetify, and TypeScript.

## Overview

PortunCmd is a modern admin dashboard template featuring:
- Vue 3 with Composition API
- Vuetify 3 for UI components
- TypeScript for type safety
- Vertical navigation layout
- Vite for fast development and building

## Documentation

All documentation is available in the [`/doc`](./doc/) folder:

- [Getting Started](./doc/GETTING_STARTED.md) - Setup and development guide

## Quick Start

### Prerequisites

- Node.js (v18 or higher)
- pnpm (package manager)

### Installation

```sh
pnpm install
```

### Development

```sh
pnpm dev
```

The development server will start at `http://localhost:5173`

### Build

```sh
pnpm build
```
